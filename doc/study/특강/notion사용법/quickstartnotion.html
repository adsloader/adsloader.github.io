2.1.3. ‍️Block은 Page를 구성하는 단위이다.
...

  • Block은 Notion을 막강하게 해주는 기능이다. 일반적인 Text를 사용할 수도 있지만 MarkDown 기반의 문법과 다양한 플러그인과 연동하는 Block들이 존재한다. 이를 통해 문서작성 → 웹페이지 작성 과 같은 강력한 결과를 만들 수 있다.
  • Markdown 문법과 그 확장된 기능을 제공한다.
  • 수많은 플러그인을 사용할 수 있다.

3. 간단한 예제 따라하기
...

아래 내용대로 따라하기로 만든다. 만들면서 Notion의 기초기능을 체화한다.

  • (1) Workspace 생성하기
    1. 좌측 상단메뉴를 누르면 사용자 정보가 popup으로 나온다
    2. 이 메뉴의 우측상단 […]를 누른다.
    3. Join or Create WorkSpace를 만든다. 만들면, 템플릿이 복사된다. 그리고 이름은 디폴트로 설정된다.

    4. Settings & members로 가서 Workspace의 이름을 바꾼다. Settings로 이동하여 Name을 나만의 워크북으로 바꾼다.

  • (2) Page 만들기
    1. 좌측메뉴에서 (+) New Page 선택
    2. 제목에 MarkDown 요약
    3. Page에 내용입력
  • (3) Markdown 문법요약한 페이지 만들기
  • (4) 공유 - 웹으로 공유하기 웹으로 공유하여 타인에게 보내본다.
  • (5) 내보내기 - HTML, PDF

4. 공유하기
...

공유하기 기능은 우측 상단의 […] 메뉴를 선택하면 [Share]와 [Publish] 기능으로 구분되어 사용할 수 있다.

  • [Publish] 기능은 웹에 게시하는 것이다. url 링크만 있으면 누구나 볼 수 있다. 그러니 모두에게 공개하는 것이 아니라면 사용하지 말 것을 권한다.
  • [Share] 기능은 지금 문서에 사용자를 초대하는 것이다. 권한을 줄 수 있다. 권한은 크게 4가지있다.
    • Full Access → 모든 기능, 다른사람 초대까지 가능
    • Can Edit → 문서 에디티 기능
    • Can Comment → 커맨트 달기 가능
    • Can View → 보기만 가능

5. 내보내기
...

pdf, html, markdown으로 내보낼 수 있다. 그러나 문서가 복잡해지거나 업로드된 파일에 한글이 있을 경우, 완벽하게 export가 되지 않는다.
