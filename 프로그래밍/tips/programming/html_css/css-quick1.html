  1. CSS란 무엇인가요?
    CSS(Cascading Style Sheets)는 HTML 요소의 스타일과 레이아웃을 정의하기 위한 스타일 시트 언어입니다. HTML은 웹 페이지의 구조를 정의하는 데 사용되고, CSS는 해당 구조에 스타일을 적용하여 디자인과 레이아웃을 제어합니다. CSS는 웹 페이지를 보다 시각적으로 표현하고 사용자 경험을 향상시키는 데 사용됩니다.
  2. CSS 사용법
    CSS는 HTML 문서에 직접 포함되거나 외부 CSS 파일을 링크하여 사용할 수 있습니다. 일반적으로 외부 CSS 파일을 사용하는 것이 유지보수에 용이합니다. 아래는 CSS를 사용하는 기본적인 방법입니다.

내부 CSS:


<!DOCTYPE html>
<html>
<head>
  <style>
    /* CSS 코드를 여기에 작성합니다. */
  </style>
</head>
<body>
  <!-- HTML 요소들 -->
</body>
</html>

외부 CSS 파일:


<!DOCTYPE html>
<html>
<head>
  <link rel="stylesheet" type="text/css" href="styles.css">
</head>
<body>
  <!-- HTML 요소들 -->
</body>
</html>
  1. CSS 예제 (10개):
    아래는 다양한 CSS 예제입니다.

(1) 배경 색상 변경:


body {
  background-color: lightblue;
}

(2) 글꼴 변경:


h1 {
  font-family: Arial, sans-serif;
}

(3) 텍스트 정렬:


p {
  text-align: center;
}

(4) 테두리 추가:


img {
  border: 1px solid black;
}

(5) 박스 그림자 추가:


div {
  box-shadow: 2px 2px 4px rgba(0, 0, 0, 0.3);
}

(6) 버튼 스타일링:


button {
  background-color: #4CAF50;
  color: white;
  padding: 10px 20px;
  border: none;
  cursor: pointer;
}

(7) 링크 스타일 변경:


a {
  text-decoration: none;
  color: blue;
}

(8) 리스트 스타일 변경:


ul {
  list-style-type: circle;
}

(9) 표 스타일링:


table {
  border-collapse: collapse;
}
td, th {
  border: 1px solid black;
  padding: 5px;
}

(10) 애니메이션 효과 추가:


@keyframes myAnimation {
  0% { opacity: 0; }
  100% { opacity: 1; }
}

div {
  animation: myAnimation 1s linear;
}

이러한 예제는 CSS의 다양한 기능을 보여줍니다. 여러분은 이를 기반으로 CSS를 배우고 더 복잡하고 흥미로운 스타일링을 구현할 수 있습니다.
