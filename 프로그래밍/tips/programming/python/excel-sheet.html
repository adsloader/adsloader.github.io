excel-sheet
title: excel sheet 이름으로 파일명 저장하기 
date: 2023-10-25 01:18:50
tags: [ChatGPT, excel]

Person:

    1. a.xls 파일이 있다.
  1. 시트가 다음과 같이 있다. [a, b, c]
  2. 시트별로 파일을 만들고 저장한다.
  3. 시트별 파일의 이름은 a_시트명.xls 이다.
    파이썬으로 소스코드를 만들어줘

ChatGPT:

  • 아래는 Python을 사용하여 주어진 요구사항에 따라 Excel 파일을 읽고 각 시트를 별도의 파일로 저장하는 코드입니다. 이 코드를 실행하려면 openpyxl 라이브러리를 설치해야 합니다.

먼저, 필요한 라이브러리를 설치하고 코드를 실행해 보겠습니다:

pip install openpyxl

그런 다음 아래의 Python 코드를 사용하여 작업할 수 있습니다:

import openpyxl

# 원본 엑셀 파일 이름
source_file = 'a.xls'

# 엑셀 워크북 열기
workbook = openpyxl.load_workbook(source_file)

# 각 시트를 별도의 파일로 저장
for sheet in workbook.sheetnames:
    # 새로운 워크북 생성
    new_workbook = openpyxl.Workbook()
    
    # 원본 시트의 데이터 복사
    source_sheet = workbook[sheet]
    new_sheet = new_workbook.active
    for row in source_sheet.iter_rows():
        for cell in row:
            new_sheet[cell.coordinate] = cell.value

    # 새로운 파일로 저장
    new_filename = f'a_{sheet}.xlsx'
    new_workbook.save(new_filename)
    print(f'{sheet} 시트를 {new_filename} 파일로 저장했습니다.')

# 원본 워크북 닫기
workbook.close()

이 코드는 원본 엑셀 파일을 열고 각 시트를 새로운 엑셀 파일로 저장합니다. 각 시트의 이름을 사용하여 파일 이름을 지정합니다. 코드를 실행하면 각 시트에 해당하는 파일이 생성될 것입니다.
