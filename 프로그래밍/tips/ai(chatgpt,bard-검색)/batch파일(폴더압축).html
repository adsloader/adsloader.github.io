batch파일(폴더압축)

Person:

    1. MacOS
  1. 기본 zip 압축프로그램 활용
  2. 현재 폴더에서 서브폴더까지 확장자 txt만 압축
  3. txt압축.zip으로 저장
    배치파일을 만들어줘

ChatGPT:

  • macOS에서 배치 파일이 아닌 쉘 스크립트를 사용하여 해당 작업을 수행할 수 있습니다. 아래는 터미널에서 사용할 수 있는 쉘 스크립트 예제입니다.

터미널을 열고 다음 단계를 따라주세요:

  1. 원하는 폴더로 이동합니다.
  2. 다음 명령어로 쉘 스크립트 파일을 생성합니다. nano compress_txt_files.sh (원하는 다른 편집기를 사용해도 됩니다)
  3. 스크립트 내용에 다음을 입력합니다:
#!/bin/bash

find . -type f -name "*.txt" -exec zip txt압축.zip {} +
  1. Ctrl + X를 누르고 Y를 입력하여 변경사항을 저장한 후 엔터키를 눌러 종료합니다.
  2. 다음 명령어를 입력하여 스크립트를 실행 가능하도록 권한을 부여합니다: chmod +x compress_txt_files.sh
  3. 이제 스크립트를 실행할 수 있습니다: ./compress_txt_files.sh

이렇게 하면 현재 폴더와 서브폴더에서 확장자가 .txt인 파일을 찾아 txt압축.zip이라는 이름으로 압축할 수 있습니다. 파일을 압축하고 싶은 폴더에서 위 스크립트를 실행하면 됩니다.

Person:

    1. Windows 10
  1. 기본 zip 압축프로그램 활용
  2. 현재 폴더에서 서브폴더까지 확장자 txt만 압축
  3. txt압축.zip으로 저장
    배치파일을 만들어줘

ChatGPT:

  • Windows 10에서는 배치 파일(.bat)을 사용하여 해당 작업을 수행할 수 있습니다. 아래는 배치 파일로 이 작업을 수행하는 방법입니다.
  1. 텍스트 편집기(메모장 등)를 열고 다음과 같은 내용을 입력합니다:

@echo off
cd /d %~dp0

echo Zipping .txt files...
powershell -Command "& {Add-Type -A 'System.IO.Compression.FileSystem'; [System.IO.Compression.ZipFile]::CreateFromDirectory('.\','txt압축.zip',([System.IO.Compression.CompressionLevel]::Optimal), $true, '^*.txt$')}"

echo Compression complete!
pause
  1. 위 코드에서 txt압축.zip 부분은 압축된 파일의 이름입니다. 원하는 이름으로 변경할 수 있습니다.
  2. 파일을 저장할 때, "모든 파일"에서 .bat 확장자로 저장하도록 선택합니다. 파일 이름 뒤에 .bat을 붙여 파일을 저장합니다. 예: compress_txt_files.bat
  3. 작성한 배치 파일(compress_txt_files.bat)을 원하는 폴더에 넣고 실행합니다. 해당 폴더와 서브폴더에서 확장자가 .txt인 파일을 찾아 txt압축.zip이라는 이름으로 압축될 것입니다.

위의 코드는 PowerShell을 사용하여 .txt 파일들을 찾고 기본 Windows 10의 Zip 기능을 이용하여 압축하는 방법입니다.
