Button 위젯의 사용법 정리

설명:
Flutter 2.7 이상부터 기존의 Button class들이 이름과 파라메터가 달라졌다. 그래서 이전의 소스들이 최신 Flutter에서는 에러가 날 수 있다. ElevatedButton(윤곽이 있는 모양), TextButton(윤곽이 없는 모양), OutlinedButton(바깥쪽 라인있는 모양)이 주로 사용된다. 그리고 모든 Button은 onPressed 파라메터에 콜백함수를 넘겨서 클릭에 대한 이벤트를 처리할 수 있다.

  • ElevatedButton(윤곽이 있는 모양), TextButton(윤곽이 없는 모양), OutlinedButton(바깥쪽 라인있는 모양)
  • FloatingActionButton(떠있는 아이콘모양), IconButton(아이콘 모양)
  • 클릭에 대한 처리는 onPressed 파라메터에 함수를 구현해서 넘김
  • child 파라메터로 Text를 생성하여 버튼에 문자열을 표시함
  • 버튼의 크기를 조절할 경우, SizedBox 안에 자식위젯(child)으로 생성
  • color는 배경색
  • shape에 버튼배경 지정가능 (ex: RoundedRectangleBorder )
  • SizedBox는 위젯의 크기를 조절할 경우 사용함

모든 위젯은 자식 위젯이 1개일 때는 child, 여러개일 때는 children이라는 필드를 사용한다.

  • FloatingActionButton은 화면에 여러개 사용할 경우, 설정값에 따라 가끔 알 수 없는 에러가 발생함.
  • Fluttertoast를 사용하기 위해서는 플러그인을 포함시켜야한다.

전체소스

  • dartpad로 실행하기
import 'package:flutter/cupertino.dart';
import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return TextTest();
  }
}


class TextTest extends StatelessWidget{
  @override
  Widget build(BuildContext context){
     return MaterialApp(
       title: "1. Text 사용법",  
       home: Scaffold(
         appBar: AppBar( title: Text("Test 입니다."),),
         body: buildTestBody())
     );
  }

  Widget buildTestBody(){
    return Column(  // 세로방향으로 위젯을 추가한다[children]. Android의 LinearLayout
      crossAxisAlignment: CrossAxisAlignment.center,
      mainAxisSize: MainAxisSize.min,
      children: <Widget>[
        // Button 예제
        ElevatedButton(
            onPressed: () {/*이벤트 핸들러*/},
            child: Text('1. ElevatedButton')),
        TextButton(onPressed: () {/*이벤트 핸들러*/}, child: Text('2. TextButton')),
        SizedBox(
            width: double.infinity, // <-- match_parent
            child: TextButton(
                 style: ElevatedButton.styleFrom(
              padding: const EdgeInsets.symmetric(horizontal: 50, vertical: 10),
              shape: RoundedRectangleBorder(
                  borderRadius: BorderRadius.circular(20))),
                onPressed: () {
                  //Fluttertoast.showToast(msg: "안녕하세요~");
                  print("console 출력");
                },
                child: Text(
                  '3. TextButton Event',
                  style: TextStyle(color: Colors.red, fontSize: 30),
                ))),
        OutlinedButton(
            style: OutlinedButton.styleFrom(
              backgroundColor: Colors.yellow,
              primary: Colors.amberAccent,
              elevation: 20, //<-- SEE HERE
              shadowColor: Colors.amber, //<-- SEE HERE
            ),
            child: Text("4. Outline Button"),
            onPressed: null,
            ), 
        
        FloatingActionButton(
          onPressed: () {},
          child: Icon(Icons.thumb_up),
          backgroundColor: Colors.pink,
        ),
        IconButton(
          icon: Icon(Icons.access_alarm),
          tooltip: 'alarm setting',
          onPressed: () {},
        )
       ],  
    );
  }
}
