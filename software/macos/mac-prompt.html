title: MacOS에서 prompt 설정하기 
date: 2023-08-14 20:36:26
tags:
category: [ChatGPT, macOS]

Person:
...

  1. 현재의 디렉토리를 보여주게 한다.
    쉘프롬프트를 알려주세요

ChatGPT:
...

  • macOS의 터미널에서 zsh 쉘을 사용할 때 현재 디렉토리를 쉘 프롬프트에 표시하는 방법은 다음과 같습니다:
  1. 터미널을 실행하고 zsh 쉘로 전환합니다.
  2. 홈 디렉토리로 이동하여 .zshrc 파일을 편집합니다. 만약 .zshrc 파일이 없다면 새로 생성합니다.
  3. .zshrc 파일에 다음과 같은 줄을 추가합니다:

PROMPT='%n@%m:%~%# '

위의 프롬프트 형식은 다음과 같이 구성됩니다:

  • %n: 현재 사용자 이름
  • @: "@" 기호
  • %m: 현재 호스트 이름
  • :: 콜론 기호
  • %~: 현재 디렉토리 경로를 홈 디렉토리를 기준으로 표시 (예: ~/Documents)
  • %#: 일반 사용자는 $ 기호로, 관리자(root)는 # 기호로 표시됩니다.
  1. .zshrc 파일을 저장하고 닫습니다.
  2. 변경 사항을 적용하기 위해 터미널에서 다음 명령어를 실행합니다:

source ~/.zshrc

이제 쉘 프롬프트에는 현재 디렉토리 경로가 표시될 것입니다.

ChatGPT:
...

  • 아래와 같이 .zshrc 파일에 다음 줄을 추가하여 원하시는 형식의 쉘 프롬프트를 설정할 수 있습니다:

PROMPT='%F{blue}%D{%H:%M:%S} %F{white}%1d %F{cyan}➜ %F{reset}'

위의 프롬프트 형식은 다음과 같이 구성됩니다:

  • %F{color}: 다음에 오는 텍스트의 색상을 설정합니다. color는 원하는 색상으로 대체됩니다. 예를 들어, blue, white, cyan은 각각 파란색, 흰색, 청록색을 나타냅니다.
  • %D{%H:%M:%S}: 현재 시간을 24시간 형식(Hour:Minute:Second)으로 표시합니다.
  • %1d: 현재 디렉토리의 부모 디렉토리를 생략하고, 현재 디렉토리 경로를 출력합니다.
  • %F{reset}: 텍스트 색상을 기본값으로 재설정합니다.

위의 설정을 .zshrc 파일에 추가한 후, 터미널에서 source ~/.zshrc 명령어를 실행하여 변경 사항을 적용합니다. 이제 쉘 프롬프트에는 현재 디렉토리 경로가 화살표 배경으로 표시될 것입니다.

PROMPT='%F{green}╭─%F{white}%1d %F{green}╰─%F{reset}'

위의 프롬프트 형식은 다음과 같이 구성됩니다:

  • ╭─: 현재 디렉토리 경로 앞에 오는 모양으로, 초록색 선과 끝점입니다.
  • %F{white}%1d: 현재 디렉토리의 부모 디렉토리를 생략하고, 현재 디렉토리 경로를 흰색으로 표시합니다.

  • %F{reset}: 텍스트 색상을 기본값으로 재설정합니다.

위의 설정을 .zshrc 파일에 추가한 후, 터미널에서 source ~/.zshrc 명령어를 실행하여 변경 사항을 적용합니다. 이제 쉘 프롬프트에는 현재 디렉토리 경로가 초록색 배경의 모양으로 표시될 것입니다.

위의 프롬프트 형식은 다음과 같이 구성됩니다:

  • %F{yellow}: 다음에 오는 텍스트의 색상을 노란색으로 설정합니다.
  • ╭─: 현재 디렉토리 경로 앞에 오는 모양으로, 노란색 선과 끝점입니다.
  • %F{white}%1d: 현재 디렉토리의 부모 디렉토리를 생략하고, 현재 디렉토리 경로를 흰색으로 표시합니다.
  • %F{yellow}╰─: 현재 디렉토리 경로 뒤에 오는 모양으로, 노란색 선과 끝점입니다.
  • %F{reset}: 텍스트 색상을 기본값으로 재설정합니다.
  • %b: 텍스트 스타일을 기본값으로 재설정합니다.

  • 프롬프트 배경색이 변경되지 않는 경우에는 몇 가지 가능한 이유가 있을 수 있습니다. 다음과 같은 점들을 확인해 보시기 바랍니다:
  1. 올바른 색상 코드를 사용하고 있는지 확인하세요: %F{yellow}를 사용하여 배경색을 설정했는지 확인해 주세요. "yellow" 대신에 다른 유효한 색상 코드를 사용하려면 %F{#RRGGBB} 형식으로 표현해야 합니다. 여기서 #RRGGBB는 해당 색상의 16진수 RGB 코드입니다.
  2. 터미널 에뮬레이터가 색상을 지원하는지 확인하세요: 색상을 지원하지 않는 터미널 에뮬레이터를 사용 중이라면 배경색이 제대로 나타나지 않을 수 있습니다. 일부 터미널 에뮬레이터는 배경색을 지원하지 않거나 설정해야 할 수도 있습니다. 터미널 프로그램이나 설정을 변경하여 색상 지원 여부를 확인해 보세요.
  3. 다른 쉘 설정과 충돌하는지 확인하세요: .zshrc 파일 외에도 다른 설정 파일(예: .bashrc, .bash_profile)이 쉘 프롬프트와 관련된 설정을 변경하고 있는지 확인해 보세요. 다른 설정 파일에서 배경색을 덮어쓰거나 재정의하는 경우 프롬프트에 영향을 줄 수 있습니다.
  4. .zshrc 파일을 제대로 로드하고 있는지 확인하세요: .zshrc 파일에 변경 사항을 저장한 후, source ~/.zshrc 명령어를 실행하여 .zshrc 파일을 로드해야 합니다. 이 명령어를 실행하지 않으면 변경 사항이 적용되지 않을 수 있습니다. 터미널에서 source ~/.zshrc 명령어를 실행하여 변경 사항을 로드해 보세요.

위의 점들을 확인하고 여전히 문제가 해결되지 않는다면, 추가적인 정보를 제공해 주시면 더 자세한 도움을 드릴 수 있습니다.
